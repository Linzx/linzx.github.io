## 博客网站：<http://gityuan.com>

博文主要包含基于**`Android 6.0`**源码的分析整理的文章，旨在从Linux最底层kernel到framework层有一个连贯地深入理解。文章计算从核心组件，从进程、内存、IO等角度对Linux/Android系统的理解，还有一些与Performance/power/debug tool等相关的文章，目前还在对源码的攻读和整理中...
